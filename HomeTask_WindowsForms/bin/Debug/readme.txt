Инструкция по установке dll(ок):

Для начала необходимо скачать нужную Вам библиотеку. Так как поисковые системы довольно категорично относятся к dll библиотекам, все файлы с 
расширением .dll мы архивируем, следовательно Вы скачаете с нашего сайта архив, в котором будет находиться необходимая вам dll(-ка). Извлекаем 
(разархивируем) dll файл из архива. Далее для разных ОС, существуют разные пути куда необходимо поместить dll(-ку):

Для Windows 95/98/Me - поместить dll файл сюда: ″C:\Windows\System″
Для Windows Windows NT/2000 - поместить dll файл сюда: ″C:\WINNT\System32″
Для Windows Windows XP, Vista, 7, 8 — поместить dll файл сюда: ″C:\Windows\System32″
Для 64-разрядных версий Windows - поместить dll файл сюда: ″C:\Windows\SysWOW64\″

В случае если файл с таким именем уже существует, на всякий случай сделайте резервную копию оригинального файла и смело соглашайтесь на замену 
старой или поврежденной dll библиотеки. Следующий шаг — перезагрузка, после чего можно пробовать запуска не работающие ранее: игры, 
программы или приложения.

Иногда бывает что библиотека не подтягивается ОС, для исправления сделайте следующее:
Открываем меню «Пуск» и выбираем «Выполнить …». В открывшемся окне, вводим команду «cmd» и жмем «Ок» или «Enter» 
(в случае если вы используете Windows ME, вводим COMMAND). Далее в терминале, пишем (без кавычек) »regsvr32 ххх.dll» 
(где ххх — это название dll файла) и жмем Enter. Перезагрузка. Если файл был скачан с нашего сайта и Вы делали все по инструкции, успех не заставит 
себя ждать, все должно получиться.
